# VA Headhunter Landing Page

A Vercel-ready one-page Vite/React landing page for VA Headhunter.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Vercel

Import this folder as a Vercel project. Vercel will use:

- Build command: `npm run build`
- Output directory: `dist`
- Framework: Vite

## Lead form

The current form is front-end only and shows a success state. Connect it to Formspree, HubSpot, Airtable, or a Vercel serverless endpoint before driving paid traffic.
